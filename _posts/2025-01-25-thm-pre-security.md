## Overview

The Pre Security path covers the foundational knowledge needed before diving into cybersecurity and hacking.

## Modules

### How The Web Works
- DNS resolution process
- HTTP/HTTPS protocols
- Web servers and clients

### Network Fundamentals
- IP addressing and subnetting
- TCP/UDP protocols
- Common ports and services

### Linux Fundamentals
- Terminal commands
- File system hierarchy
- Package management

### Windows Fundamentals
- Windows file system
- User management
- Registry and services

## Key Takeaways

- Understanding networking is the foundation of cybersecurity
- Both Linux and Windows knowledge are required
- Web technology understanding is critical for web app security
